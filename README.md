# P2P Diagrams

Приложение для отображения диаграмм на основе JSON данных.

## Установка

```bash
npm install
```

## Запуск

```bash
npm run dev
```

## Использование

1. Запустите приложение
2. Используйте кнопку "Загрузить JSON" для загрузки файла с данными
3. Формат JSON должен содержать:
   - `nodes`: массив узлов с полями `id`, `x`, `y`, `label`, `type`
   - `edges`: массив связей с полями `from`, `to`, `label`

Пример данных в файле `example-data.json`
